## 3. AI 编码不烧钱的真相：实测数据来了

HN 热帖《AI coding at home without going broke》（226↑ 207💬）给出了一个务实的视角：在家做 AI 编码其实没那么贵。

**作者实测结论：**
- 使用 API 级别的 AI 编码工具（如 Claude Sonnet 4 with extended thinking），每天编码 4-6 小时
- 月均成本约 $50-80，远低于 SaaS 订阅制的 AI 编码工具
- 关键技巧：合理设置上下文窗口、避免不必要的长对话、善用缓存

```python
# 成本控制策略示例
def estimate_monthly_cost(daily_hours=5, cost_per_hour=0.35):
    """
    估算 AI 编码月度成本
    - daily_hours: 每天编码小时数
    - cost_per_hour: 每小时 API 成本（美元），Sonnet 4 约 $0.30-0.40/h
    """
    work_days = 22  # 工作日
    monthly_cost = daily_hours * cost_per_hour * work_days
    return f"月度成本: ${monthly_cost:.2f}"

print(estimate_monthly_cost(5, 0.35))  # 月度成本: $38.50
```

这个数据点说明：AI 编码的门槛已经降到个人开发者完全可以承受的水平。但也要注意，这是 API 直连的成本，如果通过中间层（如 Cursor、Windsurf 等编辑器），订阅费会更高。
